Cargo Ship Loader

Live

Five-solver optimisation engine for loading 20 ft and 40 ft ISO containers into a Panamax-class hull. Greedy baseline, beam search, simulated annealing, Bayesian-optimised weights, and a neural position ranker — all compared on port/starboard and fore/aft balance.

Toronto OD Data Ingestor

Package

Reusable Python library for ingesting any Toronto Open Data (CKAN) package into PostgreSQL. Powers the TTC delay prediction pipeline. Supports replace and append modes with optional deduplication, and integrates with Supabase for zero-config database hosting.
